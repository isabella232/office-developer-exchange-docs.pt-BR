# <a name="fileextension-soap"></a>FileExtension (SOAP)

O **elemento FileExtension** representa uma extensão de arquivo. 
  
```XML
<FileExtension/>
```

 **cadeia de caracteres**
## <a name="attributes-and-elements"></a>Atributos e elementos

As seções a seguir descrevem os atributos, os elementos filhos e os elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum
  
### <a name="child-elements"></a>Elementos filho

Nenhum.
  
### <a name="parent-elements"></a>Elementos pai

|**Elemento**|**Descrição**|
|:-----|:-----|
|[SupportedFileExtensions (SOAP)](supportedfileextensions-soap.md) <br/> |Contém uma lista de extensões de arquivo em um local de compartilhamento de documentos.  <br/> |
   
## <a name="text-value"></a>Valor de texto

O valor de texto do **elemento FileExtension** é uma extensão de arquivo válida para o local de compartilhamento de documentos.
