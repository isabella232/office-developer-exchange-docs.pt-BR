# <a name="phone-phoneentitytype"></a>Telefone (PhoneEntityType)

O elemento do **telefone** Especifica um número de telefone único resultante de uma extração de entidade de número de telefone. 
  
```XML
<Phone>
   <Position/>
   <OriginalPhoneString/>
   <PhoneString/>
   <Type/>
</Phone>
```

 **PhoneEntityType**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

[Posição](position.md) | [OriginalPhoneString](originalphonestring.md) | [PhoneString](phonestring.md) | [tipo (string)](type-string.md)
