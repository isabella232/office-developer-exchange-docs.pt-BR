# <a name="value-bodycontenttype"></a>Valor (BodyContentType)

O elemento de **valor** Especifica o valor de um elemento **BodyContentAttributedValue** . 
  
```XML
<Value>
   <Value/>
   <BodyType/>
</Value>
```

**BodyContentType**

## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

[Valor](value.md) | [BodyType](bodytype.md)
  
### <a name="parent-elements"></a>Elementos pai

[BodyContentAttributedValue](bodycontentattributedvalue.md)
