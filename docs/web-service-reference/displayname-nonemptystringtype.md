# <a name="displayname-nonemptystringtype"></a>DisplayName (NonEmptyStringType)

O elemento de **DisplayName** contém o nome de exibição de uma nova contato de um grupo ou o nome de exibição de um novo grupo de mensagens instantâneo de mensagens instantâneas. 
  
```XML
<DisplayName></DisplayName>
```

 **NonEmptyStringType**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

Nenhum.
  
### <a name="parent-elements"></a>Elementos pai

[AddDistributionGroupToImList](adddistributiongrouptoimlist.md) | [AddImGroup](addimgroup.md) | [AddNewImContactToGroup](addnewimcontacttogroup.md) | [ImGroup](imgroup.md)
  
## <a name="text-value"></a>Text value

O valor de texto do elemento **DisplayName** é o nome de exibição de uma nova contato de um grupo ou o nome de exibição de um novo grupo de mensagens instantâneo de mensagens instantâneas.
