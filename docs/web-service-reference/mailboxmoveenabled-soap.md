# <a name="mailboxmoveenabled-soap"></a>MailboxMoveEnabled (SOAP)

O **elemento MailBoxMoveEnabled** representa o sinalizador **MailboxMoveEnabled().** O **elemento MailBoxMoveEnabled** é apenas para uso interno. Esse elemento não é usado pelos clientes. 
  
```XML
<MailBoxMoveEnabled>true | false</MailBoxMoveEnabled>
```

**Boolean**

## <a name="attributes-and-elements"></a>Atributos e elementos

As seções a seguir descrevem os atributos, os elementos filhos e os elementos pai.
  
### <a name="attributes"></a>Atributos

### <a name="child-elements"></a>Elementos filho
