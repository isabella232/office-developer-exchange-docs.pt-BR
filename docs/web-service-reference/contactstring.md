# <a name="contactstring"></a>ContactString

O **elemento ContactString** especifica o nome de exibição de um contato. 
  
```XML
<ContactString></ContactString>
```

 **cadeia de caracteres**
## <a name="attributes-and-elements"></a>Atributos e elementos

As seções a seguir descrevem os atributos, os elementos filhos e os elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum
  
### <a name="child-elements"></a>Elementos filho

Nenhum.
  
### <a name="parent-elements"></a>Elementos pai

|**Elemento**|**Descrição**|
|:-----|:-----|
|[Contact (ContactType)](contact-contacttype.md) <br/> |Especifica um contato no Armazenamento unificado de contatos.  <br/> |
   
## <a name="text-value"></a>Valor de texto

O valor de texto do **elemento ContactString** é o valor da cadeia de caracteres que especifica o nome de exibição. 
  
## <a name="remarks"></a>Comentários

Este elemento foi introduzido no Exchange Server 2013.
  
O esquema que descreve este elemento está localizado no diretório virtual do IIS que hospeda os Serviços Web do Exchange.
