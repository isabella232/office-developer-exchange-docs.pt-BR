# <a name="mailtipsaccessenabled-soap"></a>MailTipsAccessEnabled (SOAP)

O elemento **MailTipsAccessEnabled** representa o sinalizador **MailTipsAccessEnabled ()** . O elemento **MailTipsAccessEnabled** é somente para uso interno. Esse elemento não é usado por clientes. 
  
```XML
<MailTipsAccessEnabled>true | false</ MailTipsAccessEnabled>
```

 **Boolean**
