# <a name="mailtipsaccessenabled-soap"></a>MailTipsAccessEnabled (SOAP)

O **elemento MailTipsAccessEnabled** representa o **sinalizador MailTipsAccessEnabled().** O **elemento MailTipsAccessEnabled** é apenas para uso interno. Esse elemento não é usado pelos clientes. 
  
```XML
<MailTipsAccessEnabled>true | false</ MailTipsAccessEnabled>
```

 **Boolean**
