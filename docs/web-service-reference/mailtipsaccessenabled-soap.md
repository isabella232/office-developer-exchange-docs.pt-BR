# <a name="mailtipsaccessenabled-soap"></a>MailTipsAccessEnabled (SOAP)

O elemento **MailTipsAccessEnabled** representa o sinalizador **MailTipsAccessEnabled()** . O elemento **MailTipsAccessEnabled** é apenas para uso interno. Este elemento não é usado pelos clientes. 
  
```XML
<MailTipsAccessEnabled>true | false</ MailTipsAccessEnabled>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
