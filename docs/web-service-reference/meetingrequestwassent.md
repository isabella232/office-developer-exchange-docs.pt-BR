# <a name="meetingrequestwassent"></a>MeetingRequestWasSent

O elemento **MeetingRequestWasSent** indica se foi enviada uma solicitação de reunião a participantes solicitados. 
  
```xml
<MeetingRequestWasSent/>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

Nenhum.
  
### <a name="parent-elements"></a>Elementos pai

|**Elemento**|**Descrição**|
|:-----|:-----|
|[CalendarItem](calendaritem.md) <br/> |Representa um item de calendário do Exchange.  <br/> |
|[MeetingRequest](meetingrequest.md) <br/> |Representa uma solicitação de reunião no armazenamento do Exchange.  <br/> |
   
## <a name="text-value"></a>Text value

Um valor de texto que representa um valor booleano é necessário se este elemento é incluído. Um valor **true** indica que uma solicitação de reunião foi enviada. Um valor **false** indica que uma solicitação de reunião não foi enviada. 
  
## <a name="remarks"></a>Comentários

O esquema que descreve este elemento está localizado no diretório virtual do EWS do computador que está executando o MicrosoftExchange Server 2007 que tem instalada a função de servidor de Acesso para Cliente.
