# <a name="usersettings-soap"></a>UserSettings (SOAP)

O elemento **UserSettings** representa uma coleção de configurações do usuário. 
  
```XML
<UserSettings>
    <UserSetting/>
</UserSettings>
```

 **UserSettings**
## <a name="attributes-and-elements"></a>Attributes and elements

As seções a seguir descrevem os atributos e elementos filho elementos pai.
  
### <a name="attributes"></a>Atributos

Nenhum.
  
### <a name="child-elements"></a>Elementos filho

|**Elemento**|**Descrição**|
|:-----|:-----|
|[UserSetting (SOAP)](usersetting-soap.md) <br/> |Representa uma configuração de usuário único.  <br/> |
   
### <a name="parent-elements"></a>Elementos pai

|**Elemento**|**Descrição**|
|:-----|:-----|
|[Resposta SOAP)](userresponse-soap.md) <br/> |Representa uma resposta a uma solicitação de GetUserSettings para um usuário individual.  <br/> |
