# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

O **elemento ProtocolConnections** contém zero ou mais conexões de protocolo. 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```

 **ProtocolConnections**
