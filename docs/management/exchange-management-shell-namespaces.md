# <a name="exchange-management-shell-namespaces"></a>Namespaces do Shell de Gerenciamento do Exchange

Encontre informações sobre os namespaces do Shell de Gerenciamento Exchange no Exchange.
  
**Aplica-se a:** Exchange Online | Exchange Server 2013 | Office 365
  
Exchange Aplicativos shell de gerenciamento para Exchange Online, Exchange Online como parte do Office 365 ou uma versão do Exchange a partir do Exchange 2013 interagem com o ambiente Windows PowerShell por meio dos tipos expostos pelo **Namespace System.Management.Automation.** Os cmdlets Exchange Shell de Gerenciamento usam tipos de várias bibliotecas Exchange de gerenciamento. Esta seção fornece informações de referência sobre os tipos e interfaces das bibliotecas Exchange que são usadas por cmdlets Exchange Shell de Gerenciamento. 
  
O Exchange Shell de Gerenciamento contém os seguintes namespaces:
